# RockPaperScissors
🎮 A simple Java-based Rock Paper Scissors game where the user plays against the computer. The program takes user input, generates a random computer choice, and determines the winner using conditional logic. It demonstrates basic Java concepts like input handling, randomness, and decision-making.
# 🎮 Rock Paper Scissors (Java)

A basic console-based Rock Paper Scissors game built using Java.

## 🚀 Features
- User vs Computer gameplay
- Random computer choice
- Win/Lose/Draw logic
- Simple and interactive console input

## 🧠 Concepts Used
- Conditional statements (if-else)
- User input (Scanner)
- Random number generation
- Basic game logic

## ▶️ How to Run

1. Compile:
   javac RockPaperScissors.java

2. Run:
   java RockPaperScissors
